git常用命令
  • 发表于2017-10-27
  • 分类于工具

  如果要我说我能这么相对来说轻松一点看教程创博客，我得感谢我的前辈--廖雪峰老师。在我最开始了解git的时候，是他的教程一点一点带我走进命令行的世界。现在我奉上他的网址以及我的一点小总结。
  廖雪峰git教程
(一)初始化git仓库
1，初始化git仓库先使用git init命令。
2, 使用命令git add ,可反复多次添加文件，或者使用git add .命令。
3, 使用git commit -m "提示修改的消息以供以后参考",完成。 (二)工作区命令
1, 要随时掌握工作区的状态，使用git status命令。
2,status命令告诉你有文件被修改过，用git diff命令查看被修改的内容。
(三)添加远程库
1,要关联远程库使用git remote add github git@github.com:username/xxxx.git
2,关联后使用git push -u github master第一次推送master里面的所有内容。
  如果不了解，我强烈建议您去看廖雪峰的git教程，想来对你会有所帮助。
